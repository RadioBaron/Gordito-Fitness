Integritetspolicy

Senast uppdaterad: 2026-05-26


🔒 Ingen server – all data stannar på din enhet

1. Sammanfattning

Gordito Fitness samlar ingen personlig data och skickar ingenting till externa servrar. All din information (program, träningshistoria, vikt, inställningar) lagras uteslutande lokalt på din enhet via webbläsarens localStorage och IndexedDB.

2. Vilken data samlas in?

Ingen. Vi samlar varken in namn, e-post, platsdata, IP-adresser eller beteendedata. Det finns inga analysverktyg, inga trackers och inga reklamlösningar i appen.

3. Tredjepartstjänster

Appen hämtar träningsövningar från två öppna API:er för att utöka övningsbiblioteket:

• wger.de – öppen träningsdatabas. Din IP-adress kan loggas av deras server vid hämtning, precis som vid alla webbförfrågningar. Inga personuppgifter skickas.

• GitHub (free-exercise-db) – öppen övningsdatabas. Samma princip som ovan.

Google Fonts laddas för typsnitt. Google kan se att fonten begärdes, men inga app-specifika data delas.

4. Cookies

Inga cookies används. App-data lagras i localStorage och IndexedDB, inte i cookies.

5. Dina rättigheter

Du kan när som helst radera all din data via Mer → Återställ all data i appen. Det raderar all information från din enhet permanent.

6. Barn

Appen är avsedd för användare över 13 år. Vi samlar medvetet ingen data från barn.

7. Ändringar av denna policy

Eventuella ändringar publiceras på denna sida. Eftersom ingen data lämnar din enhet påverkar ändringar inte historisk insamling.

8. Kontakt
